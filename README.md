# AI Code Reviewer

A Streamlit web application that uses an LLM model to review Python code and output structured JSON issues.

## Features
- Upload or paste Python code
- Automated code review using an LLM model
- Strict JSON-formatted output
- Clean and modern Streamlit UI
- Shows raw model output for debugging
- Easy to deploy on Streamlit Cloud
